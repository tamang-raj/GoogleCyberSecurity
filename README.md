## 📖 About

This repository documents my learning journey through the **Google Cybersecurity Professional Certificate** offered by **Google** on **Coursera**.

The goal of this repository is to:

- 📚 Organize my notes from every course and module
- 🧪 Showcase hands-on labs and practical exercises
- 💻 Share scripts and projects created throughout the program
- 📈 Track my progress toward becoming a cybersecurity professional
- 🎯 Build a portfolio that demonstrates my cybersecurity knowledge and skills

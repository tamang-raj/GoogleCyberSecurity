# Question 1

### Fill in the blank:

Examples of security _____ include **security and risk management** and **security architecture and engineering**.

### Correct Answer

✅ **Domains**

### Explanation

A **security domain** is a broad area of cybersecurity knowledge and responsibility. The **(ISC)² CISSP** framework defines **eight security domains**, which provide a foundation for cybersecurity professionals.

### The Eight CISSP Security Domains

1. Security and Risk Management
2. Asset Security
3. Security Architecture and Engineering
4. Communication and Network Security
5. Identity and Access Management (IAM)
6. Security Assessment and Testing
7. Security Operations
8. Software Development Security

# Question 2

### A security professional is asked to destroy and dispose of old hard drives that include confidential customer information.

Which security domain is this task related to?

### Correct Answer

✅ **Asset Security**

### Explanation

**Asset Security** focuses on protecting an organization's physical and digital assets throughout their lifecycle.

Responsibilities include:

- Data classification
- Data storage
- Data retention
- Secure backups
- Secure disposal of storage devices
- Protecting sensitive information

Securely destroying old hard drives prevents unauthorized access to confidential customer data.

# Question 3

### Your supervisor asks you to audit user permissions for payroll data to ensure no unauthorized employees have access.

Which security domain is this audit related to?

### Correct Answer

✅ **Security Assessment and Testing**

### Explanation

The **Security Assessment and Testing** domain focuses on evaluating whether security controls are functioning correctly.

Common activities include:

- Security audits
- Vulnerability assessments
- Penetration testing
- Access reviews
- Compliance testing

Auditing user permissions ensures that only authorized personnel can access sensitive payroll information.

# Question 4

### You investigate an alert involving an unknown device connected to the company's internal network. After the investigation, you follow company procedures to implement preventative measures.

Which security domain is this scenario related to?

### Correct Answer

✅ **Security Operations**

### Explanation

The **Security Operations** domain focuses on the day-to-day protection of an organization's systems.

Responsibilities include:

- Monitoring security events
- Investigating alerts
- Responding to incidents
- Following organizational policies
- Implementing preventative measures
- Maintaining business continuity

Investigating suspicious devices and preventing future incidents are core security operations tasks.

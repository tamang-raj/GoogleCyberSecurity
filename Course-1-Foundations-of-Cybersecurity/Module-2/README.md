# 📖 Overview

This repository documents my learning journey through **Module 2: The Evolution of Cybersecurity** in the **Google Cybersecurity Professional Certificate**.

This module explores the history of cybersecurity, the evolution of cyber attacks, common attack methods, attacker motivations, and introduces the **eight CISSP security domains**, which provide a foundation for information security practices.

---

# 🎯 Learning Objectives

After completing this module, I can:

- Understand the history and evolution of cybersecurity.
- Explain how cyber attacks have changed over time.
- Identify common types of cyber attacks.
- Understand why attackers target organizations.
- Recognize different threat actors and their motivations.
- Explain the eight CISSP security domains.
- Identify which security domain applies to different security scenarios.

# 🔐 The Eight CISSP Security Domains

This module introduced the eight CISSP domains:

| Domain | Description |
|---------|-------------|
| Security and Risk Management | Governance, compliance, risk management, security policies |
| Asset Security | Protecting information and organizational assets |
| Security Architecture and Engineering | Secure system design and engineering principles |
| Communication and Network Security | Securing networks and communication systems |
| Identity and Access Management (IAM) | Authentication, authorization, and access control |
| Security Assessment and Testing | Security audits, vulnerability assessments, penetration testing |
| Security Operations | Incident response, monitoring, disaster recovery |
| Software Development Security | Secure software development lifecycle (SDLC) |

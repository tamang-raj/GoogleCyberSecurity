# 📖 Overview

This repository documents my learning journey through **Module 2: The Evolution of Cybersecurity** in the **Google Cybersecurity Professional Certificate**.

This module explores the history of cybersecurity, the evolution of cyber attacks, common attack methods, attacker motivations, and introduces the **eight CISSP security domains**, which provide a foundation for information security practices.

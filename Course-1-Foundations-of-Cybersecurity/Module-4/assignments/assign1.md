# Question 1

### What tool is designed to capture and analyze data traffic within a network?

### Correct Answer

✅ **Network Protocol Analyzer (Packet Sniffer)**

### Explanation

A **Network Protocol Analyzer**, also known as a **Packet Sniffer**, captures and analyzes network traffic traveling across a network.

Cybersecurity professionals use packet sniffers to:

- Monitor network communications.
- Troubleshoot network issues.
- Detect suspicious or malicious traffic.
- Investigate security incidents.
- Analyze network protocols.

### Popular Packet Sniffers

- Wireshark
- tcpdump
- Microsoft Message Analyzer (legacy)

# Question 2

### What type of tool uses dashboards to organize data into categories and allows analysts to identify potential security incidents as they happen?

### Correct Answer

✅ **SIEM (Security Information and Event Management)**

### Explanation

A **SIEM** platform collects, analyzes, and correlates logs from multiple systems to provide real-time visibility into security events.

SIEM tools help security teams:

- Monitor security events.
- Detect suspicious activity.
- Investigate incidents.
- Generate alerts.
- Create dashboards and reports.
- Support compliance requirements.

### Common SIEM Platforms

- Splunk
- Microsoft Sentinel
- IBM QRadar
- Google Security Operations (Chronicle)
- Elastic Security

# Question 3

### What can cybersecurity professionals use logs for?

### Correct Answer

✅ **To identify vulnerabilities and potential security breaches**

### Explanation

**Logs** record events generated by operating systems, applications, network devices, and security tools.

Security analysts use logs to:

- Detect suspicious activity.
- Investigate security incidents.
- Identify vulnerabilities.
- Track unauthorized access attempts.
- Monitor system health.
- Support forensic investigations.

### Common Types of Logs

- System logs
- Application logs
- Security logs
- Firewall logs
- Authentication logs
- Web server logs

# Question 4

### Fill in the blank:

A _____ is a manual that provides details about operational actions.

### Correct Answer

✅ **Playbook**

### Explanation

A **playbook** is a documented guide that outlines the procedures for responding to specific cybersecurity events.

Playbooks help ensure that security teams respond to incidents in a consistent, organized, and efficient manner.

A playbook typically includes:

- Incident response steps.
- Roles and responsibilities.
- Escalation procedures.
- Communication guidelines.
- Recovery actions.

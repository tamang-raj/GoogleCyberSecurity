## Questions and Answers

### 1. What tool is designed to capture and analyze data traffic within a network?

**Answer:** ✅ **Network Protocol Analyzer (Packet Sniffer)**

**Explanation:**  
A network protocol analyzer (also called a **packet sniffer**) captures and analyzes network traffic, helping security analysts monitor communications, troubleshoot network issues, and investigate suspicious activity.

---

### 2. What type of tool uses dashboards to organize data into categories and allows analysts to identify potential security incidents as they happen?

**Answer:** ✅ **SIEM (Security Information and Event Management)**

**Explanation:**  
SIEM tools collect and analyze log data in real time, organize security events using dashboards, and generate alerts for suspicious activities.

---

### 3. What can cybersecurity professionals use logs for?

**Answer:** ✅ **To identify vulnerabilities and potential security breaches**

**Explanation:**  
Logs record events that occur within systems and networks, allowing analysts to detect suspicious activity, investigate incidents, and identify vulnerabilities.

---

### 4. Fill in the blank: A _____ is a manual that provides details about operational actions.

**Answer:** ✅ **Playbook**

**Explanation:**  
A playbook is a document that outlines procedures and operational actions for responding to security incidents and performing security-related tasks consistently.

---

# 📖 Overview

This repository documents my learning progress through **Module 4: Cybersecurity Tools and Programming Languages** of the **Google Cybersecurity Professional Certificate**.

This module introduces the essential tools used by cybersecurity professionals, including Security Information and Event Management (SIEM), Intrusion Detection Systems (IDS), packet analyzers, and Endpoint Detection and Response (EDR). It also provides an introduction to Linux, SQL, and Python—three fundamental technologies widely used in cybersecurity. Finally, the module focuses on building a professional cybersecurity portfolio and preparing for an entry-level cybersecurity career.

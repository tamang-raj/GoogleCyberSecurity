This repository contains my notes, learning progress, and achievements for **Module 1** of the **Google Cybersecurity Professional Certificate** offered by Google on Coursera.

This module introduces the fundamentals of cybersecurity, explores common security concepts, explains the responsibilities of cybersecurity professionals, and discusses the essential technical and workplace skills needed for an entry-level cybersecurity career.

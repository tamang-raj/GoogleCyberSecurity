# 📖 Overview

This repository documents my learning progress for **Module 3: Protect Against Threats, Risks, and Vulnerabilities** in the **Google Cybersecurity Professional Certificate**.

This module introduces security frameworks, security controls, compliance standards, secure system design, and the ethical principles that guide cybersecurity professionals. It emphasizes protecting organizational assets through structured security practices and responsible decision-making.

---

# 🎯 Learning Objectives

After completing this module, I can:

- Explain the purpose of security frameworks.
- Differentiate between administrative, technical, and physical security controls.
- Understand secure design principles.
- Describe the relationship between security, compliance, and organizational policies.
- Apply ethical principles in cybersecurity decision-making.
- Recognize the importance of confidentiality, integrity, and availability when protecting systems.
- Understand how ethics influence cybersecurity responsibilities.

## Security Frameworks

The module introduced widely used cybersecurity frameworks, including:

- NIST Cybersecurity Framework (CSF)
- NIST Risk Management Framework (RMF)
- CIS Critical Security Controls
- ISO/IEC 27001
- Security Policies and Standards

---

## Types of Security Controls

### Administrative Controls

- Security policies
- Employee training
- Security awareness
- Risk assessments
- Incident response planning

### Technical Controls

- Firewalls
- Antivirus software
- Multi-Factor Authentication (MFA)
- Encryption
- Intrusion Detection Systems (IDS)
- Access controls

### Physical Controls

- Security guards
- Locks
- CCTV surveillance
- Badge access systems
- Biometric authentication

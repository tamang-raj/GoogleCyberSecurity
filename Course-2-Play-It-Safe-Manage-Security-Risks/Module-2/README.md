# 📚 Topics Covered

## Security Frameworks

Security frameworks provide structured guidance for managing cybersecurity risks and implementing security best practices.

Frameworks covered include:

- NIST Cybersecurity Framework (CSF)
- NIST Risk Management Framework (RMF)
- CIS Critical Security Controls
- ISO/IEC 27001 (Overview)

---

## Security Controls

Security controls are safeguards used to reduce security risks.

### Administrative Controls

- Security policies
- Employee training
- Risk assessments
- Security awareness programs
- Incident response procedures

### Technical Controls

- Firewalls
- Multi-Factor Authentication (MFA)
- Antivirus software
- Encryption
- Intrusion Detection Systems (IDS)

### Physical Controls

- Security cameras
- Badge access systems
- Locks
- Security guards
- Biometric authentication

---

## The CIA Triad

The CIA Triad is a core cybersecurity model consisting of:

### 🔒 Confidentiality

Ensures that information is accessible only to authorized users.

Examples:

- Encryption
- Access control
- Multi-Factor Authentication

---

### ✔ Integrity

Ensures data remains accurate and unaltered.

Examples:

- Hashing
- Digital signatures
- Version control

---

### 🌐 Availability

Ensures systems and data are available when needed.

Examples:

- Backups
- Redundant systems
- Disaster recovery
- High availability infrastructure

---

## NIST Cybersecurity Framework (CSF)

The six core functions of the NIST CSF are:

| Function | Purpose |
|----------|---------|
| Govern | Establish cybersecurity governance and strategy |
| Identify | Understand organizational assets and risks |
| Protect | Implement safeguards to reduce risks |
| Detect | Identify cybersecurity events quickly |
| Respond | Take action during security incidents |
| Recover | Restore systems and improve resilience |

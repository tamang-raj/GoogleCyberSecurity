# 📖 Overview

This repository documents my learning progress through **Module 2: More About Frameworks and Controls** from **Course 2 – Play It Safe: Manage Security Risks** in the **Google Cybersecurity Professional Certificate**.

This module focuses on the relationship between **security frameworks** and **security controls**, the **CIA Triad**, the **NIST Cybersecurity Framework (CSF)**, **OWASP security principles**, and the process of conducting **security audits**. It also includes a hands-on portfolio activity to perform a security audit using industry best practices.

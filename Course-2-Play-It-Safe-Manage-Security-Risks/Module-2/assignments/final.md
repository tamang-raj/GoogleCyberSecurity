## Question 2

### Question
Fill in the blank:

> An employee using multi-factor authentication to verify their identity is an example of the _____ process.

### Options
- Encryption
- Authentication
- Confidentiality
- Integrity

### ✅ Correct Answer
**Authentication**

### Explanation
**Authentication** is the process of verifying the identity of a user before granting access to systems or resources. Multi-Factor Authentication (MFA) strengthens authentication by requiring two or more verification methods.

## Question 5

### Question
Which of the following statements accurately describe the NIST Cybersecurity Framework (CSF)? *(Select all that apply.)*

### ✅ Correct Answers
- ✅ The CSF is a voluntary framework that consists of standards, guidelines, and best practices to manage cybersecurity risk.
- ✅ Investigating an incident to determine how the threat occurred, what was affected, and where the attack originated is part of the **Respond** function.
- ✅ The **Protect** function of the CSF involves implementing policies, procedures, training, and tools to mitigate threats.

### ❌ Mistake
You **did not select**:

- ✅ **The Protect function of the CSF involves implementing policies, procedures, training, and tools to mitigate threats.**

You incorrectly selected:

- ❌ **The Detect function of the CSF involves making sure proper procedures are used to contain, neutralize, and analyze security incidents.**

### Explanation
The **Detect** function focuses on identifying potential security incidents and improving monitoring capabilities.

The **Respond** function is responsible for:
- Containing incidents
- Neutralizing threats
- Investigating attacks
- Performing incident analysis
- Implementing improvements

## Question 6

### Question
A security team establishes controls, including permission settings that create multiple security layers a threat actor must bypass. Which OWASP principle does this describe?

### Options
- Principle of least privilege
- Keep security simple
- Separation of duties
- Defense in depth

### ✅ Correct Answer
**Defense in depth**

### Explanation
**Defense in Depth** uses multiple layers of security controls so that if one layer fails, additional protections remain in place.

# ❌ Mistake Review

## Question 5

### Your Mistake
You selected the **Respond** function but missed one correct option.

### Correct Answers
- ✅ CSF is a voluntary cybersecurity framework.
- ✅ Respond investigates, contains, and analyzes incidents.
- ✅ Protect implements policies, procedures, training, and security tools.

### Remember

| CSF Function | Purpose |
|--------------|---------|
| **Identify** | Understand assets and risks |
| **Protect** | Implement safeguards (training, MFA, policies, tools) |
| **Detect** | Identify security events and monitor systems |
| **Respond** | Contain, analyze, and mitigate incidents |
| **Recover** | Restore normal operations |

# Question 1

### Question

**What is the CIA triad?**

### Correct Answer

✅ **A foundational security model used to set up security policies and systems**

### Explanation

The **CIA Triad** is the core security model in cybersecurity. It guides organizations in designing security policies and systems based on three fundamental principles:

- 🔒 Confidentiality
- ✅ Integrity
- ⚡ Availability

---

# Question 2

### Question

**Which element of the CIA triad specifies that only authorized users can access specific information?**

### Correct Answer

✅ **Confidentiality**

### Explanation

**Confidentiality** ensures that sensitive information is only accessible to authorized users.

Common controls include:

- Encryption
- Passwords
- Multi-Factor Authentication (MFA)
- Role-Based Access Control (RBAC)

---

# Question 3

### Question

A security analyst discovers that certain data is inaccessible to authorized users, preventing employees from doing their jobs efficiently. The analyst fixes the application to restore timely and reliable access.

**Which element of the CIA triad does this scenario describe?**

### Correct Answer

✅ **Availability**

### Explanation

**Availability** ensures that systems, applications, and data remain accessible whenever authorized users need them.

Examples include:

- System maintenance
- Redundant servers
- Backups
- Disaster recovery plans
- Load balancing

# Question 4

### Question

**Fill in the blank:** According to the CIA triad, _____ refers to ensuring that an organization's data is verifiably correct, authentic, and reliable.

### Correct Answer

✅ **Integrity**

### Completed Sentence

> According to the CIA triad, **integrity** refers to ensuring that an organization's data is verifiably correct, authentic, and reliable.

### Explanation

**Integrity** ensures that data remains:

- Accurate
- Complete
- Authentic
- Reliable

It protects information from unauthorized modification or corruption.

Common integrity controls include:

- Hashing
- Digital signatures
- Checksums
- File integrity monitoring

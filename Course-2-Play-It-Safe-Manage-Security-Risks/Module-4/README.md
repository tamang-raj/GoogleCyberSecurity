# 📖 Overview

This repository documents my learning progress through **Module 4: Use Playbooks to Respond to Incidents** from **Course 2 – Play It Safe: Manage Security Risks** in the **Google Cybersecurity Professional Certificate**.

This module introduces **incident response playbooks**, the phases of the incident response lifecycle, and how cybersecurity analysts use **SIEM** and **SOAR** tools to efficiently detect, investigate, and respond to security incidents. It emphasizes the importance of standardized response procedures and teamwork during cybersecurity incidents.

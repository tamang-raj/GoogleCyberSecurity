# 📖 Overview

This repository documents my completion of **Course 2: Play It Safe – Manage Security Risks**, part of the **Google Cybersecurity Professional Certificate** offered by **Google** on **Coursera**.

Throughout this course, I developed a deeper understanding of **security governance**, **risk management**, **industry security frameworks**, **security controls**, **incident response**, **security auditing**, and the tools cybersecurity analysts use to detect and respond to threats.

The course combines cybersecurity theory with practical activities that reflect real-world responsibilities of entry-level cybersecurity professionals.

# Question 1 ❌

### Question
**Fill in the blank:**  
The _____ domain is focused on access and authorization to keep data secure by making sure that users follow established policies to control and manage assets.

### Your Answer
- Communication and Network Security

### ✅ Correct Answer
- **Identity and Access Management (IAM)**

### Explanation
The **Identity and Access Management (IAM)** domain focuses on controlling who can access organizational resources. It includes:

- **Identification** – Claiming an identity
- **Authentication** – Verifying identity
- **Authorization** – Determining access permissions
- **Accountability** – Monitoring user actions

Asset Security focuses on protecting assets, while Communication and Network Security focuses on securing network infrastructure.

### Explanation

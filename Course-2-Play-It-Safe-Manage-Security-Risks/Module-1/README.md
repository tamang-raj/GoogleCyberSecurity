# 📖 Overview

This repository documents my learning progress through **Module 1: More About the CISSP Security Domains** from **Course 2 – Play It Safe: Manage Security Risks** in the **Google Cybersecurity Professional Certificate**.

This module expands on the **eight CISSP security domains** and introduces the concepts of **threats, risks, and vulnerabilities**. It also covers the **NIST Risk Management Framework (RMF)** and demonstrates how cybersecurity analysts identify, assess, and manage security risks to protect organizational assets.

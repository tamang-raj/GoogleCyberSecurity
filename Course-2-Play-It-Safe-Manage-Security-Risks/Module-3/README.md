# 📖 Overview

This repository documents my learning progress through **Module 3: Introduction to Cybersecurity Tools** from **Course 2 – Play It Safe: Manage Security Risks** in the **Google Cybersecurity Professional Certificate**.

This module introduces **Security Information and Event Management (SIEM)** systems, security logs, SIEM dashboards, and the role of cybersecurity tools in detecting, monitoring, and responding to security events. It also explores common SIEM platforms and how security analysts use them to identify threats and protect organizational assets.

# 📖 Overview

This repository documents my completion of **Module 2: Network Operations and Security** from **Course 3 – Connect and Protect: Networks and Network Security** in the **Google Cybersecurity Professional Certificate**.

In this module, I explored the protocols, technologies, and security mechanisms that enable secure communication across networks. I learned how cybersecurity professionals use firewalls, VPNs, proxy servers, subnetting, and wireless security protocols to protect network infrastructure and maintain secure communication.

## Question 1

### What monitors and filters traffic coming in and out of a network?

- Uncontrolled zone
- Forward proxy server
- ✅ **Firewall**
- Domain Name System (DNS)

### Explanation

A **firewall** monitors and filters incoming and outgoing network traffic. It allows or blocks traffic based on predefined security rules, protecting networks from unauthorized access and cyber threats.

---

## Question 2

### Stateless is a class of firewall that keeps track of information passing through it and proactively filters out threats.

- True
- ✅ **False**

### Explanation

This statement is **false** because **stateful firewalls** keep track of active connections and proactively filter threats.

**Stateless firewalls:**

- Operate using predefined rules
- Do **not** track connection states
- Inspect packets individually

---

## Question 3

### Fill in the blank:

Encapsulation can be performed by a _____ to help protect information by wrapping sensitive data in other data packets.

- Firewall
- ✅ **VPN service**
- Proxy server
- Security zone

### Explanation

A **VPN service** performs **encapsulation**, wrapping encrypted data inside another packet before transmitting it across the internet. This creates a secure tunnel and protects sensitive information from unauthorized access.

---

## Question 4

### Which security zone is used to protect highly confidential information and is only accessible to employees with certain privileges?

- Demilitarized Zone (DMZ)
- Uncontrolled Zone
- ✅ **Restricted Zone**
- Management Zone

### Explanation

The **Restricted Zone** stores highly sensitive data and resources. Access is limited to authorized personnel with specific privileges and is typically protected by additional firewall rules.

---

## Question 5

### Fill in the blank:

A security analyst uses a _____ to regulate and restrict access to an internal server from the internet. This tool works by accepting traffic from external parties, approving it, and forwarding it to internal servers.

- Forward proxy server
- ✅ **Reverse proxy server**
- Controlled zone
- Port filter

### Explanation

A **Reverse Proxy Server** sits in front of internal servers and manages incoming requests from the internet. It filters, approves, and forwards legitimate traffic while hiding and protecting internal servers from direct exposure.

# Key Concepts Reviewed

| Concept | Description |
|---------|-------------|
| **Firewall** | Monitors and filters network traffic using security rules |
| **Stateless Firewall** | Filters packets individually using predefined rules |
| **Stateful Firewall** | Tracks active connections and filters traffic intelligently |
| **VPN** | Encrypts data and creates a secure tunnel over public networks |
| **Encapsulation** | Wraps encrypted data inside another packet for secure transmission |
| **Restricted Zone** | Highly secure network segment containing confidential information |
| **Reverse Proxy Server** | Protects internal servers by filtering incoming internet traffic |

---

# Quick Review

- ✅ Firewalls monitor and filter network traffic.
- ✅ Stateful firewalls track active connections; stateless firewalls do not.
- ✅ VPNs perform encapsulation to protect data during transmission.
- ✅ Restricted zones contain highly confidential information with limited access.
- ✅ Reverse proxy servers protect internal servers from external internet traffic.

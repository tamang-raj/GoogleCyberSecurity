## Question 2

### Question

Which network protocol provides a secure method of communication between clients and web servers?

### Options

- **HTTPS** ✅
- TCP
- DNS
- ARP

### Your Answer

✅ **HTTPS**

### Explanation

**HTTPS (Hypertext Transfer Protocol Secure)** encrypts communication between web browsers and web servers using **SSL/TLS**, protecting sensitive information from attackers.

## Question 3

### Question

To keep information safe from malicious actors, what security protocol can be used?

### Options

- Address Resolution Protocol (ARP)
- **Secure Sockets Layer and Transport Layer Security (SSL/TLS)** ✅
- Transmission Control Protocol (TCP)
- Domain Name System (DNS)

### Your Answer

✅ **Secure Sockets Layer and Transport Layer Security (SSL/TLS)**

### Explanation

**SSL/TLS** encrypts data transmitted over the internet and secures HTTP traffic, creating **HTTPS** connections that protect confidentiality and integrity.

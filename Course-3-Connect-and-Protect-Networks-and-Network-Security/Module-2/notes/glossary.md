# A

### Address Resolution Protocol (ARP)
A network protocol used to determine the **MAC address** of the next router or device on the communication path.

---

# C

### Cloud-based Firewalls
Software firewalls hosted and managed by a **Cloud Service Provider (CSP)** that protect cloud and on-premises resources.

### Controlled Zone
A protected subnet that separates and shields an organization's **internal network** from the uncontrolled zone (internet).

---

# D

### Domain Name System (DNS)
A networking protocol that translates **domain names** (e.g., `www.google.com`) into **IP addresses**.

---

# E

### Encapsulation
A process performed by a **VPN** that protects sensitive information by wrapping encrypted data inside additional data packets before transmission.

---

# F

### Firewall
A network security device or software that monitors and filters incoming and outgoing network traffic according to predefined security rules.

### Forward Proxy Server
A proxy server that regulates and restricts a user's access to the internet by forwarding approved requests to external servers.

---

# H

### Hypertext Transfer Protocol (HTTP)
An application-layer protocol used for communication between web browsers (clients) and web servers.

### Hypertext Transfer Protocol Secure (HTTPS)
A secure version of HTTP that encrypts communications between clients and servers using SSL/TLS.

---

# I

### IEEE 802.11 (Wi-Fi)
A family of wireless networking standards that define communication for **Wireless Local Area Networks (WLANs)**.

---

# N

### Network Protocols
A set of rules that define the **order** and **structure** of data exchanged between two or more devices on a network.

### Network Segmentation
A security technique that divides a network into smaller, isolated segments to improve performance and security.

---

# P

### Port Filtering
A firewall feature that allows or blocks network traffic based on specific **port numbers** to control unwanted communication.

### Proxy Server
An intermediary server that receives client requests and forwards them to other servers on behalf of the client.

---

# R

### Reverse Proxy Server
A proxy server that regulates and restricts internet access to an organization's internal servers by filtering incoming requests.

---

# S

### Secure File Transfer Protocol (SFTP)
A secure protocol used to transfer files between devices over a network using encrypted communication.

### Secure Shell (SSH)
A security protocol used to establish a secure remote command-line session with another computer.

### Security Zone
A segmented area of a company's network designed to protect internal resources from external networks.

### Simple Network Management Protocol (SNMP)
A management protocol used for monitoring and managing devices on a network.

### Stateful Firewall
A firewall that tracks active network connections and proactively filters traffic based on connection state and behavior.

### Stateless Firewall
A firewall that filters packets using predefined rules without tracking previous network connections.

### Subnetting
The process of dividing a network into smaller logical subnetworks called **subnets**.

---

# T

### Transmission Control Protocol (TCP)
An internet communication protocol that establishes reliable connections and enables data transmission between devices.

---

# U

### Uncontrolled Zone
The portion of a network that exists outside an organization's control, typically the **public internet**.

---

# V

### Virtual Private Network (VPN)
A network security service that encrypts internet traffic, changes a user's public IP address, and masks their virtual location to protect privacy on public networks.

---

# W

### Wi-Fi Protected Access (WPA)
A wireless security protocol that protects Wi-Fi networks by encrypting wireless communications.

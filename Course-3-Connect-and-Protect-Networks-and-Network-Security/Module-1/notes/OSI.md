# 🌐 TCP/IP Model

The **Transmission Control Protocol/Internet Protocol (TCP/IP) Model** is a conceptual framework used to visualize how data moves across a network.

Security analysts and network engineers use it to:

- Understand network communication
- Troubleshoot connectivity problems
- Detect security threats
- Analyze data transmission

---

## TCP/IP Model Layers

The TCP/IP model contains **4 layers**.

| Layer | Purpose |
|--------|---------|
| **Application Layer** | Organizes services such as file transfers and email |
| **Transport Layer** | Controls data flow and communication between devices |
| **Internet Layer** | Routes packets using IP addresses |
| **Network Access Layer** | Creates and transmits data across the physical network |

---

# 🌍 OSI Model

The **Open Systems Interconnection (OSI) Model** is a standardized **7-layer** conceptual model that explains how computers communicate over a network.

It is commonly used by network and security professionals to:

- Discuss networking problems
- Analyze security threats
- Explain how network communication works
- Troubleshoot network issues

---

## OSI Model Layers

The OSI model contains **7 layers**, starting from the application used by the user down to the physical hardware.

| Layer | Name | Purpose |
|--------|------|---------|
| **Layer 7** | Application | Direct interaction with network applications |
| **Layer 6** | Presentation | Formats and translates data |
| **Layer 5** | Session | Establishes and manages communication sessions |
| **Layer 4** | Transport | Ensures reliable data delivery |
| **Layer 3** | Network | Routes data between networks |
| **Layer 2** | Data Link | Transfers data between devices on the same network |
| **Layer 1** | Physical | Physical transmission of data through cables and hardware |

---

# 🔄 TCP/IP vs. OSI Model

| Feature | TCP/IP Model | OSI Model |
|---------|--------------|-----------|
| Number of Layers | **4** | **7** |
| Purpose | Visualizes data transmission across networks | Standardized model describing network communication |
| Used By | Network engineers and security analysts | Network and security professionals |
| Main Use | Understanding network communication and identifying disruptions | Discussing networking concepts, problems, and threats |

---

# 🤝 Similarities

Both models:

- Describe how data is transmitted across networks.
- Illustrate network processes and communication protocols.
- Divide network communication into layers.
- Help professionals troubleshoot network problems.
- Help identify security threats and communication issues.

---

# 🔑 Key Differences

| TCP/IP Model | OSI Model |
|--------------|-----------|
| 4 layers | 7 layers |
| Focuses on practical network communication | Provides a detailed conceptual framework |
| Widely used in real-world networking | Commonly used for learning, discussion, and troubleshooting |

---

# 🛡️ Why Security Professionals Use Both Models

Understanding both models helps security professionals:

- Analyze network traffic
- Identify communication issues
- Investigate security incidents
- Detect vulnerabilities
- Design secure network architectures

Different organizations may prefer one model over the other, so familiarity with both is essential.

---

# 📚 Key Terms

| Term | Definition |
|------|------------|
| **TCP/IP Model** | A four-layer framework used to visualize how data is transmitted across a network |
| **OSI Model** | A standardized seven-layer conceptual model describing network communication |
| **Application Layer** | Provides network services directly to applications |
| **Transport Layer** | Controls data transmission between devices |
| **Internet/Network Layer** | Routes data between networks |
| **Network Access Layer** | Handles physical network communication and packet transmission |
| **Physical Layer** | Deals with physical hardware and data transmission |

---

# 📝 Key Takeaways

- The **TCP/IP model** has **4 layers**.
- The **OSI model** has **7 layers**.
- Both models explain how data moves across networks.
- Both divide network communication into layers.
- The TCP/IP model is widely used to visualize data transmission.
- The OSI model provides a standardized way to understand and discuss networking.
- Security analysts should be familiar with **both models** because organizations may use either framework.

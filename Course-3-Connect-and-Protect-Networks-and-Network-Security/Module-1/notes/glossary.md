# 📖 Course 3 – Module 1 Glossary

## Google Cybersecurity Professional Certificate

### Course 3: Connect and Protect – Networks and Network Security

This glossary contains the key networking terms introduced in **Module 1: Network Architecture**.

---

## B

### Bandwidth

The maximum amount of data that can be transmitted over a network within a given period, typically measured in **bits per second (bps)**.

---

## C

### Cloud Computing

The practice of using remote servers, applications, and network services hosted on the internet instead of relying on local physical devices.

### Cloud Network

A collection of remote servers and computing resources hosted in data centers that are accessed over the internet.

---

## D

### Data Packet

The basic unit of data transmitted across a network. Data is divided into packets before being sent between devices.

---

## H

### Hub

A networking device that broadcasts incoming data to every device connected to the network without determining the intended recipient.

---

## I

### Internet Protocol (IP)

A communication protocol responsible for addressing and routing data packets between devices on a network.

### Internet Protocol (IP) Address

A unique numerical identifier assigned to a device that allows it to communicate over an IP network.

---

## L

### Local Area Network (LAN)

A network that connects devices within a limited geographic area, such as a home, office, or school.

---

## M

### Media Access Control (MAC) Address

A unique hardware identifier assigned to a network interface card (NIC) that identifies a device on a local network.

### Modem

A networking device that connects a local network to an Internet Service Provider (ISP), enabling internet access.

---

## N

### Network

A group of interconnected devices that communicate and share data and resources.

---

## O

### Open Systems Interconnection (OSI) Model

A seven-layer conceptual framework used to describe how data travels across a network and how different networking technologies interact.

---

## P

### Packet Sniffing

The process of capturing and analyzing network packets to monitor traffic, troubleshoot issues, or investigate security events.

### Port

A logical communication endpoint that allows applications and services to send and receive data across a network.

---

## R

### Router

A networking device that connects multiple networks and forwards data packets between them based on IP addresses.

---

## S

### Speed

The rate at which data is transmitted or received over a network, typically measured in bits per second (bps).

### Switch

A networking device that forwards data only to the intended destination device within a local network using MAC addresses.

---

## T

### TCP/IP Model

A four-layer networking model that describes how data is transmitted across interconnected networks using the TCP/IP protocol suite.

### Transmission Control Protocol (TCP)

A connection-oriented protocol that establishes a reliable communication session between devices and ensures data is delivered accurately and in order.

---

## U

### User Datagram Protocol (UDP)

A connectionless communication protocol that transmits data without establishing a dedicated connection, prioritizing speed over reliability.

---

## W

### Wide Area Network (WAN)

A network that connects devices across large geographic regions such as cities, countries, or continents.

---

## 📚 Summary

This module introduced the fundamental networking concepts that every cybersecurity professional should understand, including:

* Computer networks
* Network devices
* LAN and WAN
* IP addressing
* MAC addressing
* TCP/IP model
* OSI model
* Data packets
* Cloud networking
* Network communication protocols

These concepts provide the foundation for understanding network security, monitoring network traffic, and protecting organizational infrastructure.

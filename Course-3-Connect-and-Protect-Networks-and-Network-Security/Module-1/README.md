# 🌍 TCP/IP Model

The TCP/IP model consists of four layers.

| Layer | Purpose |
|--------|---------|
| Application | User services and network applications |
| Transport | End-to-end communication and reliability |
| Internet | Routing and IP addressing |
| Network Access | Physical transmission of data |

---

# 📚 OSI Model

The Open Systems Interconnection (OSI) Model consists of seven layers.

| Layer | Description |
|--------|-------------|
| 7. Application | Network services for users |
| 6. Presentation | Data formatting and encryption |
| 5. Session | Session management |
| 4. Transport | Reliable communication |
| 3. Network | Routing and logical addressing |
| 2. Data Link | Frame transmission |
| 1. Physical | Physical media and signals |

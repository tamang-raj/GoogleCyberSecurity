# 📖 Overview

This repository documents my learning progress through **Module 1: Network Architecture** from **Course 3 – Connect and Protect: Networks and Network Security** in the **Google Cybersecurity Professional Certificate**.

This module introduces the fundamentals of computer networking, including network components, cloud networking, communication models, IP addressing, and the TCP/IP and OSI models. These concepts form the foundation for understanding how devices communicate and how cybersecurity professionals secure modern networks.

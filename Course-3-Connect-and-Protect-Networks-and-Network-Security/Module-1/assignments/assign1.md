# Question 1

### ❓ Question

**To connect an entire city, the most effective network type would be a local area network (LAN).**

- True
- **False** ✅

### ✅ Correct Answer

**False**

### 💡 Explanation

A **Local Area Network (LAN)** covers a **small geographic area**, such as:

- Home
- School
- Office

A **Wide Area Network (WAN)** connects devices across **large geographic areas**, including:

- Cities
- States
- Countries

The **Internet** is the largest example of a **WAN**.

# Question 2

### ❓ Question

**A security professional wants to ensure information is being broadcast to every computer on their organization's network. What device should they investigate?**

- Router
- **Hub** ✅
- Internet
- Modem

### ✅ Correct Answer

**Hub**

### 💡 Explanation

A **Hub** broadcasts incoming data to **every connected device**, regardless of the intended recipient.

Unlike a switch, a hub does **not** determine the correct destination for data.

# Question 3

### ❓ Question

**What are some benefits of switches?** *(Select all that apply.)*

- ✅ They only pass data to the intended destination.
- ❌ They automatically install device-protection software.
- ✅ They can improve network performance.
- ✅ They control the flow of traffic.

### ✅ Correct Answers

- They only pass data to the intended destination.
- They can improve network performance.
- They control the flow of traffic.

### 💡 Explanation

Switches are more intelligent than hubs because they:

- Send data only to the intended device.
- Reduce unnecessary network traffic.
- Improve network performance.
- Help increase network security.

# Question 4

### ❓ Question

**Fill in the blank: The practice of using servers, applications, and network services that are hosted on the internet is called _____ computing.**

- uploadable
- connected
- website
- **cloud** ✅

### ✅ Correct Answer

**Cloud**

### 💡 Explanation

**Cloud computing** is the practice of using remote servers, applications, storage, and network services hosted on the internet instead of relying on local physical infrastructure.

Cloud computing provides:

- Scalability
- Cost savings
- High availability
- On-demand resources
- Flexible access from anywhere

# 📚 Key Concepts Reviewed

- **LAN** → Small geographic area (home, office, school)
- **WAN** → Large geographic area (city, country, Internet)
- **Hub** → Broadcasts data to every connected device
- **Switch** → Sends data only to the intended destination
- **Cloud Computing** → Internet-based servers, applications, and network services

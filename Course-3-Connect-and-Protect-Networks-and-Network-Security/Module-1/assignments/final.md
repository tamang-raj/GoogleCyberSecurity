## Question 2 ❌

### Question

Which network device connects multiple networks together?

### Options

- A modem
- A switch
- A hub
- **A router** ✅

### Your Answer

❌ Incorrect

### Correct Answer

✅ **A router**

### Explanation

A **router** connects **multiple networks** and forwards data packets between them using IP addresses.

Other devices:

- **Hub** → Broadcasts data to every connected device.
- **Switch** → Connects devices within the same LAN and forwards data only to the intended device.
- **Modem** → Connects a local network to the Internet.

### Options

- Speed

## Question 2

### Question

Which of the following is an example of an **IPv4** address?

### Options

- 2001:0db8:85a3:0000:0000:8a2e:0370:7336
- **172.16.254.1** ✅
- 00-B1-D0-63-C2-26
- 25, 443, 20

### Correct Answer

**172.16.254.1**

### Explanation

An **IPv4 address** consists of **four decimal numbers (0–255)** separated by periods.

Example:

```text
172.16.254.1
```

The first option is an **IPv6 address**, the third is a **MAC address**, and the fourth contains **port numbers**.

## Question 3

### Question

What type of address is assigned by an internet service provider (ISP) and connected to a geographic location?

### Options

- Private IP address
- MAC address
- WAN address
- **Public IP address** ✅

### Correct Answer

**Public IP Address**

### Explanation

A **public IP address** is assigned by an **Internet Service Provider (ISP)** and identifies a network on the public internet.

All devices on the same local network typically share the same public IP address through **Network Address Translation (NAT)**.

## Question 4

### Question

**Fill in the blank:** A switch uses a MAC _____ to direct data packets to the correct device.

### Options

- Geographic location
- Home network
- Public address
- **Address table** ✅

### Correct Answer

**Address table**

### Explanation

A network **switch** maintains a **MAC address table**, which maps MAC addresses to connected devices. This allows the switch to forward data only to the intended destination, improving both efficiency and security.

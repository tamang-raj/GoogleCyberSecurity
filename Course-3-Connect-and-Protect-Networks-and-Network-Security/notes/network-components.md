# 🌐 What is a Network?

A **network** is a collection of connected devices that communicate and exchange data.

### Examples
- Home Wi-Fi network
- Office network
- School network
- The Internet

Devices on a network include:
- Computers
- Smartphones
- Servers
- Printers
- Smart TVs
- IoT Devices

---

# 📡 Device Communication

Every device communicates using unique addresses.

## IP Address
- Logical address
- Identifies a device on a network
- Used for routing data

## MAC Address
- Physical hardware address
- Unique to every network interface
- Used within local networks

Communication can occur through:

- Ethernet cables
- Wireless (Wi-Fi)

---

# 🌍 Types of Networks

## Local Area Network (LAN)

A network covering a small geographic area.

Examples:
- Home Wi-Fi
- Office
- School

### Characteristics

- Fast
- Private
- Limited coverage

---

## Wide Area Network (WAN)

A network spanning large geographical areas.

Examples:
- Internet
- Company branch offices
- Country-wide networks

### Characteristics

- Large coverage
- Connects multiple LANs
- Slower than LAN

---

# 🖥️ Network Devices

## Hub

A hub sends data to **every connected device**.

### Characteristics

- Broadcasts all traffic
- Less secure
- Older technology

---

## Switch

A switch intelligently sends data only to the intended device.

### Advantages

- Better security
- Faster communication
- Reduces unnecessary traffic

---

## Router

Routers connect different networks.

### Responsibilities

- Routes packets
- Connects LAN to WAN
- Uses IP addresses

---

## Modem

A modem connects your local network to the Internet.

### Function

- Converts signals
- Provides internet access

---

## Firewall

A firewall protects networks by monitoring incoming and outgoing traffic.

### Functions

- Blocks unauthorized access
- Filters traffic
- Protects internal devices

---

# ☁️ Network Virtualization

Many networking devices can now be virtual instead of physical.

Examples include virtual:

- Routers
- Switches
- Firewalls

### Benefits

- Lower cost
- Scalability
- Easier management
- Cloud integration

# 💻 Client-Server Model

## Client

- Browser

---

## Server

Provides services or resources.

Examples

- Web server

Network architecture describes how devices connect and communicate.

- Clients
- Servers
- Routers
- Switches
- Firewalls
- Internet connections

Understanding network architecture helps security analysts:

- Locate vulnerabilities
- Protect assets
- Monitor traffic
- Develop security strategies

# 🎯 Key Takeaways

- Networks connect devices for communication.
- Devices communicate using IP and MAC addresses.
- LANs cover small areas, while WANs connect large regions.
- Switches are more secure and efficient than hubs.
- Routers connect networks and direct traffic.
- Firewalls protect networks from unauthorized access.
- Client-server architecture powers most modern networks.
- Network diagrams help security professionals understand and secure infrastructure.

# ☁️ What is Cloud Computing?

**Cloud computing** is the delivery of computing services—including storage, servers, networking, software, and databases—over the internet instead of relying entirely on physical hardware located on-site.

Organizations use **Cloud Service Providers (CSPs)** to access these resources whenever needed.

---

# 🏢 On-Premise vs. Cloud Computing

| On-Premise Infrastructure | Cloud Computing |
|---------------------------|----------------|
| Hardware is located within the organization | Resources are hosted by a Cloud Service Provider (CSP) |
| Organization manages all infrastructure | CSP manages infrastructure |
| High upfront hardware costs | Lower initial investment |
| Limited scalability | Easily scalable |
| Requires in-house maintenance | Maintenance handled by the CSP |

---

# 🌐 Cloud Service Providers (CSPs)

Cloud Service Providers offer organizations access to computing resources without requiring them to own or maintain physical infrastructure.

Examples of services include:

- Virtual servers
- Storage
- Databases
- Networking
- Security services
- Application hosting

---

# ☁️ Cloud Service Models

## Software as a Service (SaaS)

Provides ready-to-use software over the internet.

### Examples

- Gmail
- Microsoft 365
- Google Docs

### Benefits

- No installation required
- Accessible from anywhere
- Automatic updates

---

## Infrastructure as a Service (IaaS)

Provides virtual computing infrastructure.

Includes:

- Virtual machines
- Storage
- Networking
- Operating systems

### Benefits

- Flexible infrastructure
- Full control over virtual resources
- Pay only for what you use

---

## Platform as a Service (PaaS)

Provides a complete development environment.

Developers receive:

- Development tools
- Databases
- Runtime environments
- Deployment services

### Benefits

- Faster application development
- No infrastructure management
- Simplified deployment

---

# 🌍 Hybrid Cloud Environment

A **Hybrid Cloud** combines:

- On-premise infrastructure
- Cloud services

This allows organizations to:

- Maintain control over sensitive systems
- Reduce operational costs
- Increase flexibility
- Expand computing resources when needed

---

# 💻 Software-Defined Networks (SDNs)

A **Software-Defined Network (SDN)** is a virtual network managed through software instead of relying entirely on physical networking devices.

Virtual devices include:

- Virtual routers
- Virtual switches
- Virtual firewalls

Many SDNs are hosted within a Cloud Service Provider's data center.

---

# ✅ Benefits of Software-Defined Networks

- Simplified network management
- Greater flexibility
- Faster deployment
- Improved scalability
- Reduced hardware requirements

---

# 🚀 Benefits of Cloud Computing

## Reliability

Cloud providers deliver highly available services with minimal downtime.

---

## Cost Savings

Organizations avoid purchasing expensive hardware and only pay for the services they use.

---

## Scalability

Resources can be increased or decreased based on business needs.

---

## Flexibility

Employees can securely access applications and data from almost anywhere.

---

## Reduced Maintenance

Cloud providers manage hardware maintenance, updates, and infrastructure.

# 🔐 Cloud Computing in Cybersecurity

Cloud technologies help security teams by:

- Protecting cloud-hosted data
- Managing secure access
- Monitoring cloud infrastructure
- Improving availability
- Supporting disaster recovery
- Strengthening business continuity

---

# 📚 Key Terms

| Term | Definition |
|------|------------|
| Cloud Computing | Delivering computing services over the internet |
| Cloud Service Provider (CSP) | Company that provides cloud infrastructure and services |
| SaaS | Software delivered over the internet |
| IaaS | Virtual computing infrastructure provided as a service |
| PaaS | Cloud platform for developing and deploying applications |
| Hybrid Cloud | Combination of on-premise infrastructure and cloud services |
| Software-Defined Network (SDN) | A virtual network managed through software instead of physical devices |
| Virtual Router | Software-based router that directs network traffic |
| Virtual Switch | Software-based switch used in virtual networks |
| Virtual Firewall | Software firewall that filters network traffic in cloud environments |

---

# 🎯 Key Takeaways

- Cloud computing delivers computing resources over the internet.
- Cloud Service Providers (CSPs) manage infrastructure and services.
- SaaS provides software, IaaS provides infrastructure, and PaaS provides development platforms.
- Hybrid cloud environments combine on-premise systems with cloud services.
- Software-Defined Networks (SDNs) replace many physical networking devices with virtual ones.
- Cloud computing offers reliability, scalability, flexibility, and cost savings.
- Cloud technologies play a major role in modern cybersecurity.
